## 如何运行
### 方式 1：本地运行（推荐）
1. 用 VS Code 打开项目文件夹  
2. 安装并使用 **Live Server**  
3. 右键 `index.html` → **Open with Live Server**

### 方式 2：GitHub Pages 在线运行
当你在 GitHub 仓库里开启 GitHub Pages 后，用浏览器打开：
`https://kafeihukfh.github.io/p5-sketch/`

 
> `https://kafeihukfh.github.io/p5-sketch/`

## 操作说明
- 移动鼠标：改变“聚光灯”位置（吸引粒子）
- 鼠标点击：产生涟漪波
- 按 `1` / `2`：切换两种强度模式（柔和/高能）
- 按 `C`：清屏
- 按 `S`：保存截图

## 使用的工具
- p5.js
- HTML / JavaScript
- GitHub Pages（用于在线展示）
